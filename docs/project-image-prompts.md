# Project Image Prompts

The current portfolio UI renders one main `aspect-video` visual per project card and project detail page. This document defines one hero image prompt per project so the same image can be reused in both places.

## Shared Direction

- Aspect ratio: `16:9`
- Style: premium SaaS/product marketing image, cinematic lighting, polished UI render, realistic but slightly stylized
- Composition: clean focal point, strong depth, high contrast, minimal clutter, portfolio-worthy presentation
- UI treatment: modern dashboard or app interface where relevant, layered panels, glassmorphism only when it fits the product
- Do not include: readable paragraphs, brand logos, watermarks, stock-photo people, extra hands, distorted text, messy backgrounds
- Preferred output: dark or moody scene with project-specific accent colors matching the portfolio gradients

## 01. AI Lead Qualification & CRM Auto-Sync

**Suggested filename:** `ai-lead-qualification-hero.webp`

**Prompt:**  
Create a cinematic product hero image for an AI lead qualification platform. Show a modern website chatbot widget having a smart business conversation on the left and a CRM pipeline dashboard on the right, with structured lead fields, lead score, budget, company size, and webhook sync indicators flowing between them. Use a sleek blue-violet palette, soft neon highlights, premium SaaS dashboard styling, layered floating panels, subtle data lines, and a dark polished background. Make it feel like inbound leads are being qualified and pushed into a sales system automatically. No readable UI copy, no logos, no people.

## Reusable Suffix

Append this suffix to any prompt when using an image model that benefits from extra steering:

`ultra-detailed product render, premium SaaS hero shot, realistic interface composition, dramatic soft lighting, depth, reflections, crisp edges, no watermark, no distorted UI, no gibberish text`
